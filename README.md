# 粤志通 2026 志愿方案复核公测版

> 本工具仅对用户自行录入的广东高考志愿候选方案进行整理和辅助分析，不提供官方招生数据，不构成录取预测或志愿填报承诺。

## 当前定位

粤志通当前版本是“广东高考志愿方案复核助手 MVP”。用户先通过广东省普通高考志愿填报辅助系统、《广东省2026年普通高等学校招生专业目录》及高校招生章程筛选院校专业组，再将候选方案录入本系统。

系统负责：

- 比较候选院校专业组的近三年最低排位；
- 做冲刺、偏冲、稳妥、偏稳、保底、数据不足分层；
- 检查学费、民办/中外合作、调剂、城市偏好、专业排斥和数据缺失风险；
- 提供官方核对清单；
- 导出打印友好网页、CSV、JSON和简洁文本。

系统不会：

- 自动生成院校代码、专业组代码、招生计划、最低分或最低排位；
- 使用 `demo.db` 的231条演示数据作为真实推荐；
- 登录或提交广东省官方志愿填报系统；
- 收集身份证号、考生号、志愿系统密码、手机验证码或家庭住址；
- 输出精确录取概率或保证录取。

## 快速开始

```bash
npm install
npm run dev
```

访问 `http://localhost:3000`。

## 验证

```bash
npm run lint
npm run typecheck
npm test
npm run build
```

## 数据策略

- `ALLOW_DEMO_DATA=false`
- `official.db` 和 `demo.db` 保留但不参与当前 MVP 主流程
- 2023-2025导入、校验、审计脚本保留，后续可继续建设官方数据能力
- 当前公测主流程只分析用户在浏览器本地录入的候选方案

## 必须核对

最终填报前，必须以《广东省2026年普通高等学校招生专业目录》、广东省教育考试院志愿填报系统及高校2026年招生章程为准。

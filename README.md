# 粤志通 2026 志愿方案复核公测版

广东高考志愿候选方案复核与历史投档数据研究工具。

- GitHub 仓库：https://github.com/2135195482-hash/yuezhitong
- GitHub Pages：https://2135195482-hash.github.io/yuezhitong/
- Vercel 备用预览：https://yuezhitong-ldqdgr2yf-2135195482-hashs-projects.vercel.app

## 当前定位

本工具不是广东省官方志愿填报系统，也不替代广东省普通高考志愿填报辅助系统。用户应先通过官方系统、《广东省2026年普通高等学校招生专业目录》和高校 2026 年招生章程核实代码、计划、专业、选科、收费和录取规则，再把候选院校专业组放进本工具做第二层复核。

首页核心流程：

1. 查询广东省教育考试院已公开的 2023-2025 普通类本科批院校专业组投档历史数据。
2. 手工录入或批量粘贴用户自己的候选志愿草稿。
3. 在浏览器本地完成梯度、费用、调剂、专业方向、数据缺失和官方核对清单分析。

## 发布形态

当前公开版是 Next.js 静态导出：

- 不需要账号系统
- 不需要数据库
- 不需要 API Key
- 不读取 `demo.db`
- 不把用户志愿草稿上传到服务器
- 用户数据默认保存在浏览器 `localStorage`

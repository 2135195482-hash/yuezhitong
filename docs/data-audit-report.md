## 抽样核验发现

随机抽取18条记录（覆盖2023/2024/2025三个年份、物理类/历史类两个科类、来自至少15所院校），全部18条均为演示数据。

### 关键问题

1. **来源URL为占位地址**: 所有记录的 `sourceUrl` 均为 `https://eea.gd.gov.cn/gkgs/`，这是一个通用公告列表页，不是具体文件的精确URL。
2. **发布日期为假**: 官方发布日期统一为"YYYY-07-20"，这是占位日期。
3. **分数为推算值**: 最低分通过 `score_from_rank()` 函数从排名反向推算，而非来自官方投档数据。
4. **专业组代码为推断**: 专业组代码（201/202/203/101/102）基于常见模式手动构造。
5. **招生计划为随机数**: 招生计划数由 `random.choice()` 生成。
6. **同一专业组跨年复用代码**: 如中山大学"202"专业组在2023/2024/2025三个年份复用同一代码。
7. **2025年数据实际不存在**: 2025年广东高考录取数据在当前日期(2026年6月)尚未完成全部招生录取工作，2025-07-20的发布日期是提前假设的。

## 审计建议

1. **立即**将所有演示数据从正式数据库迁移到独立的演示库
2. 下载并导入广东省教育考试院官方投档数据的真实文件
3. 每条记录保存精确的官方文件URL和文件哈希
4. 双重验证通过后方可标记为 `verified`
5. 当前年度未公布的数据不得填入
